Opdracht 16: Voor deze opdracht is het handig om in te lezen over methods van String . Aan de linkerkant staat een menubalk. Onder de categorie methods staan een lijst met functies die je met een String kan aanroepen. Maak een eigen voorbeeld van de volgende functies en licht toe wat de functie doet in het blok: 1. charAt()

    Returns the character at the specified index (position)

    2. concat()

    Joins two or more strings, and returns a new joined strings

    3. slice()

    Extracts a part of a string and returns a new string

    4. split()

    Splits a string into an array of substrings

    5. substr()

    Extracts the characters from a string, beginning at a specified start position, and through the specified number of character
