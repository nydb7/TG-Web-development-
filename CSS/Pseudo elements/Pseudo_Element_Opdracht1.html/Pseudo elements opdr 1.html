Opdracht 1 Kun je enkele voorbeelden geven van pseudo-elementen (minimaal 4)? Beschrijf met je eigen woorden wat deze pseudo-elementen inhouden.

Pseudo-element
  • :first-letter

    • Voegt een speciale stijl toe aan de eerste letter van een tekst

  • :first-line

    • Voegt een speciale stijl toe aan de eerste regel van een tekst

  • :before

    • Voegt enige inhoud toe voor een element

  • :after

    • Voegt enige inhoud toe na een element
