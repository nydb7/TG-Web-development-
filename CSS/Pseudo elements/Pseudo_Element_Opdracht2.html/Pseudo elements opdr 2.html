Opdracht 2 Er zijn 14 verschillende pseudo elementen. Je hebt in de voorgaande opdracht reeds 4 kunnen benoemen. Kun je de andere 10 ook benoemen en aangeven wat ze precies doen?

  • ::selection

    • Selecteert het stukje dat de websitebezoeker zelf heeft geselecteerd

  • backdrop

    • renders a box the size of the viewport beneath any element being presented in fullscreen-mode

  • cue

    • Is used to style captions and other cues with video text tracks

  • cue-region

    • Is used to style captions of a specific region

  • grammar-error

    • Geeft aan wanneer er een grammaticale fout is

  • marker

    • Selects the marker box of a list item so you can style the bullets/numbers

  • part

    • Selects any elemets that has a matching part attribute

  • placeholder

    • Selects the placeholder text in an input or textarea

  • slotted

    • Selects any element placed in a slot inside a html page

  • spelling-error

    • Used to style a text segment which the user agent has flagged as incorrectly spelled
