Opdracht 3 Er zijn verschillende selectors, bijvoorbeeld;

  • type selectors
  • descendant selectors
  • class selectors
Maak van de bovengenoemde selectors voorbeelden. Geef in je antwoord ook aan wat de voordelen zijn van elke selector.

  • type selectors

    • Deze selector selecteert alle elementen die tot de genoemde type behoren

    p { background-color: skyblue; }

  • descendant selectors

    • Deze selector stelt je in staat om de stijl alleen aan een element toegevoegd, wanneer deze een descendent is van één of meer andere elementen.

    ul li { background-color: yellow; }

  • class selectors

    • Deze selector zorgt ervoor dat je een element met een specifieke class kan selecteren

    .skyblue { background-color: yellow; }
