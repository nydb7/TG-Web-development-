De target attributen vertellen waar de link geopend moet worden

  • _blank

    • Wanneer je een de target blank gebruikt opent de webiste in een nieuw tablat

    TechGrounds website opent in top

  • _self

    • Wanneer je een de target self gebruikt opent de webiste in hetzelfde tablat

    TechGrounds website opent in self

  • _parent

    • Wanneer je een de target parent gebruikt opent de webiste in de parent frame

    TechGrounds website opent in parent

  • _top

    • Wanneer je een de target top gebruikt opent de webiste in de gehele body van de windom

    TechGrounds website opent in top
