6. We hebben nu geleerd hoe je dropdown list, radio buttons en checkboxes maak. Dit zijn 3 verschillende manieren om de gebruiker een keuze te laten maken. Benoem voor dropdown list, radio buttons en checkboxes een situatie waar je in kan gebruiken en leg dat uit?

  • dropdown list

    • Je gebruikt een droplist wanneer je wilt dat de website bezoeker zijn voorkeur kenbaar maakt

  • Radio buttons

    • Je gebruikt een radio button wanneer de website bezoeker moet kiezen tussen twee of meer opties. Waarin deze opties elkaar wederzijds niet uitsluiten

  • checkboxes

    • Je gebruikt een checkbox wanneer de website bezoeker zelf\ mag kiezen hoeveel opties hij wilt kiezen
